# Collapse toolbar in Mishnat Yosef
A Tampermonkey script that adds a collapse button to the toolbar on the Mishnat Yosef website.
When clicked, it hides both the toolbar and the shopping cart, giving you more space to view the products themselves.

## How to install:

Install Tampermonkey from [here](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
.

Then go to:
https://github.com/ShlomoHeller/Collapse-toolbar-in-Mishnat-Yosef/raw/refs/heads/main/Collapse%20toolbar.user.js

Click Install, and enter the Mishnat Yosef [website](https://mishnatyosef.org/)\.
